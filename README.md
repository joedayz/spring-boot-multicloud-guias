# Spring Boot Multicloud — Guías gratuitas

Material educativo **100 % gratuito** para aprender Spring Boot en profundidad: APIs, microservicios, arquitectura hexagonal/limpia, GraphQL, Spring Cloud, nube (AWS y Azure) y calidad de código.

Este repositorio está pensado para acompañar vídeos de YouTube: cada guía es una **ruta clara** con pasos numerados, comprobaciones y enlaces a documentación oficial.

## Cómo usar este repo

1. Sigue el [índice de guías](docs/00-indice.md) en orden o salta al módulo que necesites.
2. Marca los checklists al terminar cada sección.
3. Si publicas vídeos, puedes enlazar la guía correspondiente en la descripción del vídeo.

## Requisitos generales

- **Java** 17 o 21 (LTS recomendado)
- **IDE**: IntelliJ IDEA, VS Code o Eclipse
- **Docker** (para bases de datos y microservicios locales)
- Cuentas gratuitas en **AWS** y **Azure** (capas free tier) cuando llegues a los módulos de nube
